# ASD

## Requirements

* Python 3
* Pipenv `pip3 install pipenv`
* Pytest `pip3 install pytest`
* VRPLIB `pip install vrplib`



## Setup

pipenv install --dev


## Development

### Run tests

pipenv run test

pytest -v tests/

### Run linter

pipenv run lint
